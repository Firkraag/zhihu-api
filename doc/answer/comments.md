# answer.comments

```javascript
answer.comments([page])
```

Get comments under this answer.

For example:

```javascript
// https://www.zhihu.com/question/20395761/answer/44292338
api.answer(13111581)
    .comments()
    .then(console.log)
    .catch(console.trace)
```
